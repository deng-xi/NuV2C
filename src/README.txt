***********************************
Tool Name:: v2c
Purpose:: Verilog to C Translator
***********************************

****************************************************************************
Input:  Synthesizable Verilog RTL
Output: Equivalent ANSI-C program
****************************************************************************

****************************************************************************
Directory Structure:
v2c/  Contains source code of the Translator 
*****************************************************************************


*****************************************************************************
Command to run v2c:
./v2c main.v --module top main.c
*****************************************************************************

*****************************************************************************
Output: ANSI-C file is generated in output named main.c 
******************************************************************************
