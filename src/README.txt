Purpose:: Verilog to C Translator
***********************************

****************************************************************************

****************************************************************************
Directory Structure:
NuV2C/  Contains source code of the Translator
*****************************************************************************

Command to run NuV2C:
./NuV2C source_files.v --module <name of top module> output_c_file.c
*****************************************************************************

*****************************************************************************
